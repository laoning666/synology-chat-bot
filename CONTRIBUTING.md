# Contributing to Synology Chatbot

First off, thanks for taking the time to contribute!

## Reporting Bugs

This section guides you through submitting a bug report for Synology Chatbot.

- **Use a clear and descriptive title** for the issue to identify the problem.
- **Describe the verification steps** in specific detail.
- **Include logs** via `docker logs`.

## Suggesting Enhancements

This section guides you through submitting an enhancement suggestion for Synology Chatbot.

- **Use a clear and descriptive title** for the issue to identify the suggestion.
- **Provide a step-by-step description of the suggested enhancement** in as much detail as possible.
- **Explain why this enhancement would be useful** to most Synology Chatbot users.
